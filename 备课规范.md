# 备课材料生成规范

> 本文档是生成备课材料时的通用规则，适用于所有学生、所有科目、所有周次。

---

## 一、目录结构

```
{学生姓名}/
├── 学生/{周次}/     ← 打印给学生用的材料（每节课1份）
└── 教师/{周次}/     ← 教师用操作手册（每节课1份）
```

- 按**周次**分文件夹（格式如 `08`、`09`），与学校进度对应
- **每个周次目录下只有1个文件**：学生拿1份，教师拿1份

---

## 二、文件命名规则

```
{编号}-{科目}-{内容}（学生版）.md
{编号}-{科目}-{内容}（教师版）.md
```

| 元素 | 说明 | 示例 |
|------|------|------|
| 编号 | 两位数，按上课顺序排 | `01`、`02` |
| 科目 | 中文 | `语文`、`道法`、`历史` |
| 内容 | 中文，具体知识点或本课主题 | `期中备考冲刺`、`语法与阅读理解` |
| 类型 | `（学生版）` / `（教师版）` | 括号包裹 |

### 示例

```
黄涵松/学生/08/
  01-语文-语法与阅读理解（学生版）.md       ← 学生唯一拿到的
  02-道法-社会规则与责任（学生版）.md
  03-历史-秦汉隋唐（学生版）.md
黄涵松/教师/08/
  01-语文-语法与阅读理解（教师版）.md       ← 包含学生版全文 + 答案解析教案
  02-道法-社会规则与责任（教师版）.md
  03-历史-秦汉隋唐（教师版）.md
```

### 编号规则
- 编号 = 上课顺序，学生版和教师版同编号对齐
- 一对一多科目时按科目拆分为多个文件，每个科目一个编号

---

## 三、学生版规范

### 核心原则
- **学生每次课只拿1份文档**（每科1份，多科多份）
- **绝对不能包含任何答案**
- 打印出来就是一份完整的课堂练习/学习材料

### 格式要求
- 顶部写明：姓名、日期、限时、满分
- 格式清晰，排版适合打印（A4纸）
- 末尾留空白区域（答题空间）
- 题目编号清晰，方便教师说"看第X题"

### 一份学生版文档的典型结构

```
# 科目 · 本课主题

> 姓名：XXX　　日期：______　　限时：__分钟　　满分：__分

---

## 第一部分：小测（X分钟，X分）

（题目内容……）

## 第二部分：知识讲解/阅读理解（X分钟，X分）

（题目内容……）

## 第三部分：练习（X分钟，X分）

（题目内容……）
```

### 禁止出现
- ❌ 参考答案
- ❌ 解析或提示
- ❌ "打印时删除此部分"等提示
- ❌ 任何教师视角的内容

---

## 四、教师版规范

### 核心原则
- **教师每次课只拿1份文档**（每科1份）
- **教师版 = 学生版全文 + 答案解析 + 教学指令**
- 打开就能用，是完整的操作手册

### 结构（严格按此顺序）

```
# 科目 · 本课主题（教师版）

> 学生：XXX　　日期：XXXX-XX-XX　　时长：__分钟
> 学校进度：……

---

## 【学生材料 · 全文】

> 以下为学生版完整内容，方便对照学生正在做哪道题。

（完整复制学生版内容，一字不差）

---

## 【课前准备】

（课前要做什么……）

---

## 【教学流程】

### 【0-X分钟】第一部分：小测

**指令：** "XXX，先给你X分钟做一个小测。"

> ⏱ 计时。观察：……

---

### 讲评：第1题

**答案：** XX

**讲解要点：**
- ……

**如果学生答对了：** "很好，你来说说为什么选XX？"
**如果学生答错了：** "你选了XX对吧？来看看……"

---

### 【X-X分钟】第二部分：……

（依次类推，覆盖学生版每一道题）

---

## 【课后安排】

（作业、家长沟通要点……）
```

### 每道题的讲解必须包含

| 要素 | 说明 |
|------|------|
| 答案 | 明确的标准答案 |
| 题目原文 | 完整复制题目，方便对照 |
| 逐项解析 | **选择题必须逐项分析每个选项为什么对/为什么错，不能只解析正确选项** |
| 讲解要点 | 为什么是这个答案，错项的陷阱在哪里 |
| 分支判断 | 学生答对 / 答错 / 部分对，分别怎么处理 |
| 过渡指令 | 讲完这道题后给学生说的下一句话 |

### 选择题解析格式（严格遵守）

必须用**表格逐项分析**，每个选项都要说明判断结果和理由：

```
| 选项 | 内容 | 判断 | 分析 |
|------|------|:----:|------|
| A | 选项内容 | ✅/❌ | 为什么对/为什么错 |
| B | 选项内容 | ✅/❌ | 为什么对/为什么错 |
| ... | ... | ... | ... |
```

⚠️ **严禁只解析正确选项或只说"ABD排除"。** 老师需要能照着解析把每个选项都讲清楚。

### 教师版关键要求
- **学生材料全文在最前面**，方便翻阅对照，知道学生正在做哪道题
- **每个教学环节有明确的指令**：`你说："……"`，精确到给学生说什么
- **每个环节有计时**：`⏱ 计时`
- **每个环节有观察要点**：观察学生什么表现
- **分支判断要覆盖**：全对、错几道、完全不会，分别怎么讲
- **语气口语化**，不要教案腔
  - ❌ "教师应引导学生思考"
  - ✅ "涵松，来了？先给你5分钟做一个小测"

### 知识要点处理
- 如果有需要学生记的公式/框架/口诀，直接写在教学流程中，教师口头讲解
- 不再单独生成"要点文件"，学生版中如果有知识卡片区域可以直接放

---

## 五、题目难度要求

### 选择题
不考单纯记忆（"XX是什么"），多考理解、辨析、综合：
- **材料型**：给场景让学生判断
- **比较型**：对比两个概念
- **组合型**：多个要素混搭
- **辨析型**：表面合理实则有错

### 简答题
每套至少2道，每道4分，贴合一模考法：
- 辨析题（判断+分析+总结）
- 情境分析题（给场景，用知识分析）
- 比较题（两个事物的异同）
- 史论结合题（举例子+分析原因）
- **必须附评分标准**（每个得分点标明分值）

---

## 六、内容生成原则

### 三科共用2小时的取舍
- 每科约40分钟，只聚焦 **1个核心知识点**
- 结构：小测5min → 核心15-20min → 休息5min → 抢进度10-15min → 练习+总结

### 知识点呈现
- 能用表格就不用文字
- 能给公式就不给思路
- 易混淆点用对比表格
- 简答题给"先写什么→再写什么→最后写什么"的框架

### 安全
- Token、密码等敏感信息不入库

---

## 七、文件联动检查

每次修改文件后，按以下清单检查：

| 改了什么 | 检查什么 |
|---------|---------|
| 学生版内容 | 教师版中的"学生材料全文"是否同步更新 |
| 文件名 | 学生版和教师版编号是否对齐 |
| README | 文件列表与实际文件是否一致 |
