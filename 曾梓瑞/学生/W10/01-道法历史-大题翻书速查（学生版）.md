## 二、道法大题：速查360翻书实战（25分钟）

### 步骤卡片

| 步骤 | 操作 | 例子 |
|------|------|------|
| ①读题 | 划出题目问什么（意义/原因/做法/启示） | "分析意义" |
| ②抓词 | 从材料中圈出核心词 | "光盘行动""文明餐桌""减少浪费" |
| ③翻书 | 用速查360的关键词索引找对应知识点 | 翻"文明""节约资源""社会责任" |
| ④套模板 | 按题型套答题框架 | 意义题=对个人+对社会+对国家 |
| ⑤组织答案 | 分点作答，每点"知识点+材料结合" | ①有利于...②体现了...③促进了... |

### 道法题型速查表

| 题型 | 关键词 | 答题框架 |
|------|--------|---------|
| 意义题 | 意义/作用/影响/好处 | ①对个人 ②对社会 ③对国家 |
| 原因题 | 为什么/原因/必要性 | ①现实需要 ②理论依据 ③发展趋势 |
| 做法题 | 怎么做/建议/措施 | ①国家(立法) ②社会(监督) ③个人(行动) |
| 启示题 | 启示/警示/教训 | ①认识到... ②应该做到... ③避免... |
| 体现题 | 体现了/说明/反映 | ①知识点是什么 ②材料怎么体现的 |
| 辨析题 | 是否正确/谈谈看法 | ①判断 ②理由(正+反) ③总结 |

## 三、历史大题：速查360翻书实战（25分钟）

### 历史题型速查表

| 题型 | 关键词 | 答题框架 |
|------|--------|---------|
| 背景题 | 背景/条件/原因 | ①政治 ②经济 ③思想文化 |
| 内容题 | 内容/措施/表现 | 按材料逐条概括，不遗漏 |
| 意义题 | 意义/影响/作用 | ①对当时 ②对后世 |
| 比较题 | 相同点/不同点/对比 | 逐项对比：背景/内容/结果/性质 |
| 评价题 | 评价/看法/认识 | ①积极作用 ②局限性 ③总结 |
| 启示题 | 启示/教训/经验 | ①从成功的经验 ②从失败的教训 |

### 历史翻书技巧

| 技巧 | 说明 |
|------|------|
| 时间定位 | 先看题目里的时间/朝代→翻对应章节 |
| 人物定位 | 看人名→翻人物相关事件 |
| 事件定位 | 看事件名→翻事件的背景/内容/意义 |
| 速查索引 | 用速查360的关键词目录→直接翻到对应页 |
| 小册子 | 万维历史小册子的答题模板→套用对应题型 |
