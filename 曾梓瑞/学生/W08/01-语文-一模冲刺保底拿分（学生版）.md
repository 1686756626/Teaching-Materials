### （二）现代文阅读（34分）

阅读下面的文章，完成12-16题。

**父亲的半壶酒**

①父亲一辈子不喝酒，除了过年。但每年除夕，他都会从柜子深处翻出那壶酒——一个掉了漆的军用水壶，里面永远只装半壶。

②"爸，这壶酒多少年了？"我小时候问过。父亲摸了摸壶身说："比你岁数大。"

③后来我才知道，那是爷爷留给父亲的。爷爷年轻时在部队当通讯兵，有一年冬天执行任务，零下三十多度的山里，他把仅剩的半壶酒分给了受伤的战友，自己硬扛着冻了回来。后来爷爷复员回乡，每年除夕都往这壶里倒半壶酒，喝一口，敬一口——一口敬战友，一口敬那些没回来的人。

④爷爷走后，父亲接过了这个习惯。他不会喝酒，每年除夕只抿一小口，然后把壶放回柜子。我注意到他每次放下壶的时候，都会用袖子把壶口擦得很干净。

⑤我上大学那年，第一次离家在外过除夕。宿舍楼空荡荡的，我一个人坐在床上给父亲打电话。电话接通的时候，我听见那边很安静。

⑥"吃了没？"父亲问。

⑦"吃了，食堂的饺子。"我说。其实我泡了一碗方便面。

⑧"你等一下。"父亲放下电话，过了好一会儿才回来，声音有点哑："爸刚才喝了一口。"

⑨那天晚上我在宿舍哭了很久，不是因为想家，是因为突然明白了那半壶酒里装的是什么——不是酒，是一个人对另一个人说不出的话。

⑩后来我在城里成了家，每年除夕都回去。父亲的头发一年比一年白，但那个习惯从未改变——翻出壶，倒半壶，抿一口，擦壶口，放回去。

⑪今年除夕，我把儿子抱到父亲面前。父亲看着孙子，笑得眼睛眯成一条缝。晚上，他又翻出了那个壶。

⑫"爸，"我说，"今年让我倒吧。"

⑬父亲看了我一眼，把壶递了过来。我往壶里倒了半壶酒，然后学着父亲的样子，抿了一小口。

⑭酒很辣，很苦，但我没有皱眉。

## 三、作文考前冲刺（13分钟）

### 第一步：拿到题目，先找"大词"

| 题目里的关键词 | 对应什么主题 | 选什么物件/场景 |
|---|---|---|
| 爱、温暖、细节、陪伴 | 亲情 | 家里的旧东西（工具箱、围裙、旧照片） |
| 成长、改变、不再害怕 | 成长/突破 | 和挑战有关的东西（号码布、故事书） |
| 感谢、朋友、互助 | 友情 | 别人给的东西（错题本、小纸条） |
| 老师、鼓励、信任 | 师生情 | 教室里的东西（粉笔盒、笔记本） |
| 传统、传承、老手艺 | 文化传承 | 老人做的东西（剪纸、竹编） |
| 善良、温暖、陌生人 | 陌生人善意 | 食物、小礼物（包子、雨伞） |
| 科技、变化、远程 | 科技温情 | 手机、屏幕、视频 |
| 坚持、热爱、不放弃 | 兴趣坚守 | 乐器、画笔、运动装备 |
| 责任、担当、服务 | 责任担当 | 自己做的东西（便民箱、志愿牌） |
| 自然、生命、发现 | 自然感悟 | 植物、动物 |

### 第二步：如果题目是"大词"（如"中国名片""创造""风景"）

**万能接法：我的小故事 + 拔高一句**

| 题目大词 | 怎么接你的小故事 | 结尾升华句 |
|---|---|---|
| 中国名片/风景/代表 | 我认真做的一件事 | "每个人认真做好自己，就是一张名片" |
| 创造/创新 | 我用新方法解决老问题 | "小创造也是创造，人人都是创造之人" |
| 责任/担当 | 我主动承担的一件小事 | "责任不分大小，从身边做起" |
| 美好/温暖 | 别人对我/我对别人的善意 | "美好就在身边，每个普通人都了不起" |
| 成长/改变 | 我从害怕到勇敢 | "改变就是成长，每一步都算数" |
| 坚守/坚持 | 我一直没放弃的事 | "坚持本身就是意义" |
| 传承/传统 | 老人教我的东西 | "传承从一个人开始" |
| 突破/超越 | 我做到了以前做不到的事 | "超越自己就是最大的突破" |

### 第三步：用"物件三步法"写

不管什么题目，文章结构都一样：

**开头（50字）：** 一个物件 → 引出回忆
> "阳台角落的旧木箱上，落着一层薄灰……"

**中段（400字）：**
> 1. 日常小事（100字）— 这个物件平时什么样
> 2. 突发转折（150字）— 发生了什么特别的事
> 3. 我的改变（150字）— 我从中学到/做到什么

**结尾（100字）：** 物件的意义 + 扣题升华
> "它让我明白……" + 紧扣题目关键词
