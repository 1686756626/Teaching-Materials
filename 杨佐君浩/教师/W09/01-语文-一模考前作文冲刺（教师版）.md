# 一模考前作文冲刺 · 教师版

## 课程信息
- **课段**：一模考前最后一课（黄涵松 4/12 周六 14:00-16:00 / 龙其乐+杨佐君浩 4/13 周日 14:00-16:00）
- **时长**：120分钟
- **目标**：让学生考场上拿到任何题目都能在3分钟内定好素材和结构

## 学生画像
- 三人作文水平相近，均 38/50（76%）
- 已学过十二大主题记叙文模板
- 核心问题：B（拿到题目不知道套哪个模板）+ C（细节写不具体）
- 正在用《备战2026年中考作文考前抢分模板》教学

## 时间分配

```
0-15min    速查表讲解（学生版第一步+第二步）
15-35min   练习一：速配训练（3道真题，只练选素材）
35-50min   练习二：素材挖掘（每人挖2-3个真实素材）
50-55min   休息
55-110min  练习三：当场写一篇完整作文
110-120min 点评 + 修改
```

## 教学要点

### 速查表讲解（15min）

**核心要传达的一句话：**
> "考场作文不是考你写得多好，是考你能不能快速把题目和你的故事接上。"

**讲法：**
1. 先问学生："去年中考题《我也是一张中国名片》，你会写什么？" → 大概率答不上来
2. 拿速查表演示：题目大词"中国名片"→ 接法"认真做的一件事" → 比如"我练吉他三年" → 结尾"每个人认真做好自己，就是一张名片"
3. 强调：**不需要想完美的素材，只需要快速配对**

### 速配训练（20min）

**操作方式：**
- 每道题给学生3分钟思考，只填表不写全文
- 填完后让学生说自己的选择
- 老师重点看两点：物件够不够具体、故事是不是真的
- 如果学生选的物件太虚（比如"我选一本书"），追问"哪本书？什么封面？什么时候买的？"

**参考答案（不是标准答案，只是示例）：**

#### 第1题：《突破自我》

| 步骤 | 示例 |
|---|---|
| 题目大词 | 突破、超越 |
| 选什么物件 | 跑道上的号码布 / 画满叉的试卷 / 第一次上台的照片 |
| 真实故事 | 我从害怕800米到跑完1500米 / 我数学从不及格到及格 |
| 结尾升华 | "突破不是赢别人，是赢自己。快放弃时再坚持一步，就是突破。" |

#### 第2题：《科技让生活更美好》

| 步骤 | 示例 |
|---|---|
| 题目大词 | 科技、变化、美好 |
| 选什么物件 | 手机里和爷爷奶奶的视频通话记录 / 小时候的照片对比 |
| 真实故事 | 因为爸爸出差没法回老家，但通过视频给爷爷奶奶过生日 |
| 结尾升华 | "科技不是冷冰冰的屏幕，是让爱不再受距离限制的桥梁。" |

#### 第3题：《我也是一张"中国名片"》

| 步骤 | 示例 |
|---|---|
| 题目大词 | 中国名片、代表、风景 |
| 选什么物件 | 琴盒里的旧吉他 / 磨损的毛笔 / 用了三年的错题本 |
| 真实故事 | 我坚持练吉他三年，虽然比赛没获奖，但一直在认真练 |
| 结尾升华 | "陈佩佳用售票手速成为中国名片，我用三年的坚持成为自己的一张名片。每个认真对待自己热爱的人，都是一道风景。" |

### 素材挖掘（15min）

**操作方式：**
- 把练习二的6个问题一个个问
- 学生回答时老师追问细节："当时什么天气？""你说了什么？""那个东西现在在哪？"
- **目标：每人至少挖出2个有细节的真实素材**
- 把素材当场写在学案上，这就是他们的"考场弹药库"

**追问技巧：**
- 学生说"我奶奶会剪纸" → 追问"她剪的什么？现在还剪吗？谁教她的？"
- 学生说"我害怕演讲" → 追问"是什么时候的事？在哪？后来怎么克服的？"
- 学生说"有同学帮我补习" → 追问"怎么帮的？写了什么？那天天气怎么样？"

### 当场写作（55min）

**操作方式：**
1. 让学生从练习二挖出的素材里选一个
2. 从3道真题里选一个最配的题目
3. 用"物件三步法"写完整作文（600字以上）
4. 写的过程中不要打断

**点评重点：**
- 开头有没有具体物件（不是"在我记忆中"这种虚开头）
- 中段有没有转折（不能平铺直叙）
- 细节够不够具体（有没有颜色、声音、触感、温度）
- 结尾有没有扣题升华（不是泛泛而谈"我学到了很多"）

## 常见问题预案

| 学生表现 | 怎么办 |
|---|---|
| "我没有什么特别的故事" | 每个人都有，从家常小事开始："你爸有什么用了很久的东西？""你最近一次哭是什么时候？" |
| 物件选得太虚 | 追问三个具体问题：什么颜色？什么材质？现在在哪？ |
| 写出来的细节像流水账 | 让学生闭眼回忆那个场景，说一个他看到的/听到的/摸到的 |
| 结尾升华太假/太假大空 | 回到题目关键词，用"不是……而是……"句式重新写 |
| 实在写不完 | 降低要求：只写开头+中段转折+结尾，中间可以简略 |

## 课后布置

- 把练习二剩余的问题也写完（6个素材全部挖出来）
- 从中选2个素材，各写一个详细版细节段落（200字/个）
- 考前每天看一遍速查表，看到题目就练"3秒配对"
