## 【85-100分钟】小测讲评 + 中国现代史收尾（15分钟）

**动作：** 收小测，对答案。

你说："自己对一下。"

### 重点讲第2题（最容易错）

你说："第2题，新中国成立的标志是什么？开国大典对吧。那我问你，新中国成立等于社会主义制度建立吗？"

> 等他回答。

你说："不等于！这是中考最容易搞混的。"

你在纸上画：

```
1949 新中国成立 → 新民主主义社会（不是社会主义！）
        ↓
1956 三大改造完成 → 社会主义制度建立 → 进入社会主义初级阶段
```

你说："记住这个时间点：**1949≠1956**。1949是建国，1956才是社会主义。一模选择题大概率会挖这个坑。"

### 讲改革开放（3分钟）

你说："学校讲到改革开放对吧？记住一句话——**1978年十一届三中全会 = 改革开放的开端**。"

你在纸上写：

```
1978 十一届三中全会 → 改革开放开端
1980 四大经济特区 → 深珠汕厦
1992 南方谈话 → 改革开放新阶段
1997 香港回归
1999 澳门回归
```

你说："这几个时间点背下来，中国现代史的选择题基本够用了。"

### 讲一国两制（2分钟）

你说："一国两制注意一个陷阱——最初是为谁提出的？"

> 等他回答。

你说："**台湾！不是香港！** 先为台湾提出，先在香港实践。考试问'一国两制最初为谁提出'选台湾，问'首先在哪里实践'选香港。别搞反了。"

## 【100-115分钟】抢进度：世界史核心考点（15分钟）

你说："学校中国史快讲完了，但世界史大概率来不及过一轮。一模世界史肯定会考，我带你快速过一遍核心考点，有个印象就行。"

### 工业革命（5分钟）

你在纸上画一个对比表：

```
第一次 vs 第二次工业革命

          第一次              第二次
时间      18世纪60年代        19世纪70年代
标志      蒸汽机              电力
时代      蒸汽时代            电气时代
发明      火车、轮船          电灯、电话、汽车、飞机
领先国家  英国                美国、德国
```

你说："这张表背下来。一模考工业革命基本就是从这张表里出题。记住最核心的：蒸汽→电气，英国→美德。"

### 两次世界大战（5分钟）

你说："两次世界大战对比着记，效率翻倍。"

你在纸上写：

```
一战 vs 二战

         一战            二战
时间     1914-1918       1939-1945
根本原因 帝国主义发展不平衡  法西斯侵略扩张
导火索   萨拉热窝事件      德国突袭波兰
转折点   凡尔登战役        斯大林格勒战役
性质     帝国主义战争      反法西斯战争
```

你说:"三组对应记住：萨拉热窝→凡尔登，突袭波兰→斯大林格勒。导火索对应转折点。"

### 冷战（3分钟）

你说："二战之后就是冷战。记住一条线——"

你在纸上写：

```
1947 杜鲁门主义 → 冷战开始
1949 北约vs华约 → 两极格局
1962 古巴导弹危机 → 冷战高潮
1991 苏联解体 → 冷战结束、两极格局终结
```

你说："冷战后的趋势记住三个词：**多极化、全球化、区域集团化**。考到就这三个方向。"

### 其他重要考点（2分钟）

你说："最后几个零散的记住："

- 文艺复兴：以人为本，反对神权
- 新航路开辟：世界连为一体（哥伦布发现美洲、麦哲伦环球航行）
