# 语文教案 · 名著阅读

> 学生：龙其乐　　日期：2025-04-05　　时长：40分钟
> 学校进度：名著阅读——讲了开头，过了一遍作者

---

## 【课前准备】
- 打印 `01-语文-名著阅读小测（原卷）` 给学生做
- 打印 `02-语文-名著阅读要点`，课后留给他
- 课前看一遍 `01-语文-名著阅读小测（解析）`

---

## 【0-5分钟】名著小测（5分钟）

你说："其乐，来了？先做个小测，5分钟，考名著的。"

> ⏱ 计时，观察他做题速度和犹豫的题目。

---

## 【5-20分钟】小测讲评 + 作者速记（15分钟）

**动作：** 收小测，对答案。

你说："来，自己先对一下答案。"

> 等他对完。

**重点讲第2题和第3题（最容易混的）：**

你说："第2题，水浒传绰号。林冲是'豹子头'，这个你答对了没有？水浒传的人物绰号必须背下来，一模大概率出选择或者简答。我给你一个速记——"

你在纸上写：

```
林冲=豹子头  武松=行者  李逵=黑旋风
鲁智深=花和尚  宋江=及时雨  吴用=智多星
```

你说："这六个是最高频的，先记住。其他人物下次再补。"

"第3题，《红星照耀中国》作者是埃德加·斯诺，美国记者。这个容易跟别人搞混，记住——斯诺=美国=红星。"

---

## 【20-35分钟】名著内容精讲（15分钟）

你说："学校把作者过了一遍，但作者只是入门，考试考的是**内容和人物**。今天我带你快速过四本最重要的名著。"

### 《西游记》（4分钟）

你说："《西游记》不用多讲，你肯定熟。但考试不考你故事好不好看，考的是**主题和人物性格**。"

你在纸上写：

```
孙悟空：叛逆 → 成熟（大闹天宫 vs 取经路上）
主题：不畏艰难、追求理想
```

你说："记住这两个点就够了——孙悟空的性格变化，和整本书的主题。"

### 《水浒传》（5分钟）

你说："水浒传考点最密集。首先，整本书的核心主题就四个字——**官逼民反**。记住这四个字，所有理解题都围绕它答。"

你说："然后是人物。水浒传最常考的是**林冲**，为什么？因为他性格变化最明显。"

你在纸上写：

```
林冲：逆来顺受 → 忍让 → 被陷害 → 被逼反抗 → 上梁山
（性格变化 = 官逼民反的典型体现）
```

你说："考试如果问'林冲的性格变化说明了什么'，你就答——说明在黑暗的社会中，好人被逼无奈也会反抗，体现了官逼民反的主题。"

### 《红星照耀中国》（2分钟）

你说："这本书记住三件事：作者是美国记者斯诺；写的是陕北苏区的采访；核心人物有毛泽东、周恩来、朱德、彭德怀。"

### 《昆虫记》（2分钟）

你说："《昆虫记》记住：法布尔写的，法国人；既是科学书也是文学书；核心主题是对生命的尊重和热爱。"

### 《简·爱》（2分钟）

你说："简·爱记住一个词——**独立自尊**。如果考人物分析，你就围绕这四个字写：独立思考、自尊自爱、追求平等。"

---

## 【35-38分钟】答题模板（3分钟）

你说："最后，名著题有万能答题模板，你记一下。"

你在纸上写：

```
人物分析：XX是一个（性格）的人。从（情节）可以看出。
情节概括：谁 + 什么情况下 + 做了什么 + 结果怎样
人物比较：相同点+ 不同点
```

你说："这三个模板套进去，名著题基本不会丢分。"

---

## 【38-40分钟】布置作业

你说："今天讲了四本名著，回去做两件事：
1. 把这份名著要点好好看一遍，重点看人物表
2. 《西游记》和《水浒传》各选一个经典故事，用自己的话概括一下，下次课我检查"
