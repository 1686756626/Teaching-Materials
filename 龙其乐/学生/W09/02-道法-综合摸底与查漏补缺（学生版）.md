## 一、宪法

### 为什么宪法是根本法？

| 对比 | 宪法 | 普通法律 |
|------|------|---------|
| 内容 | 规定国家最根本、最重要的问题 | 规定某一方面的问题 |
| 效力 | **最高法律效力**，普通法律不得与宪法相抵触 | 效力低于宪法 |
| 制定修改 | **最严格的程序** | 普通程序 |

> 一句话：宪法是国家的根本法，是其他法律的立法基础和立法依据。

### 公民基本权利

| 类别 | 具体权利 |
|------|---------|
| 政治权利 | 选举权、被选举权、言论自由 |
| 人身自由 | 人身自由权、人格尊严权、住宅不受侵犯 |
| 社会经济 | 财产权、劳动权、物质帮助权 |
| 文化教育 | 受教育权 |

### 公民基本义务

| 义务 | 说明 |
|------|------|
| 遵守宪法和法律 | 最基本的义务 |
| 维护国家利益 | 安全、荣誉和利益 |
| 依法服兵役 | |
| 依法纳税 | |

## 二、权利与义务的关系

> **权利和义务是统一的，不可分割的。**

| 观点 | 说明 |
|------|------|
| 权利义务相互依存 | 有权利就有义务，有义务就有权利 |
| 权利义务相互促进 | 行使权利有助于履行义务，履行义务有助于享有权利 |
| 不能只享权利不尽义务 | 也不应只尽义务不享权利 |

---

## 三、基本经济制度

| 组成部分 | 内容 |
|---------|------|
| 所有制 | 公有制为主体、多种所有制经济共同发展 |
| 分配制度 | 按劳分配为主体、多种分配方式并存 |
| 体制 | 社会主义市场经济体制 |

> 公有制经济：国有经济（主体）+ 集体经济 + 混合所有制中的国有/集体成分
> 非公有制经济：个体经济 + 私营经济 + 外资经济（重要组成部分）
