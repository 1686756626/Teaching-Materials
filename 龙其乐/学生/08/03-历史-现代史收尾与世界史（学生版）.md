## 一、中国现代史（学校已学，第8周收尾）

### 新中国成立（1949-1956）

| 时间 | 事件 | 意义 |
|------|------|------|
| 1949.10.1 | 开国大典 | 中华人民共和国成立 |
| 1950-1953 | 抗美援朝 | 保家卫国，提高了国际威望 |
| 1950-1952 | 土地改革 | 废除封建土地制度 |
| 1953-1956 | 三大改造 | **1956年完成，社会主义制度建立** |
| 1954 | 第一部宪法颁布 | 社会主义类型的宪法 |

### 改革开放（1978至今）

| 时间 | 事件 | 内容 |
|------|------|------|
| **1978年底** | **十一届三中全会** | **改革开放的开端** |
| 1978 | 家庭联产承包责任制 | 安徽凤阳小岗村首创，农村改革 |
| 1980 | 设立经济特区 | 深圳、珠海、汕头、厦门 |
| 1984 | 城市经济体制改革 | 国有企业改革 |
| 1988 | 海南经济特区 | 最大的经济特区 |
| 1992 | 邓小平南方谈话 | 改革开放进入新阶段 |
| 2001 | 加入WTO | 对外开放新高度 |

### 一国两制

| 事件 | 时间 |
|------|------|
| "一国两制"构想提出 | 20世纪70年代末（针对台湾） |
| 中英联合声明 | 1984年 |
| 香港回归 | **1997年7月1日** |
| 澳门回归 | **1999年12月20日** |
| 海峡两岸交流 | 1987年台湾开放探亲后逐步发展 |

## 二、世界史核心考点（学校还没到，抢进度用）

### 文艺复兴与新航路开辟

| 事件 | 时间 | 核心人物 | 内容 |
|------|------|---------|------|
| 文艺复兴 | 14-17世纪 | 但丁、达芬奇、莎士比亚 | 以人为本，反对神权 |
| 新航路开辟 | 15-16世纪 | 哥伦布、麦哲伦 | 世界开始连为一体 |

### 工业革命

| | 第一次工业革命 | 第二次工业革命 |
|--|-------------|-------------|
| 时间 | 18世纪60年代-19世纪中期 | 19世纪70年代-20世纪初 |
| 标志 | 蒸汽机的广泛应用 | 电力的广泛应用 |
| 发明 | 珍妮机、蒸汽机、火车、轮船 | 电灯、电话、汽车、飞机 |
| 国家 | 英国领先 | 美国和德国领先 |
| 影响 | 进入"蒸汽时代" | 进入"电气时代" |

### 两次世界大战

| | 第一次世界大战 | 第二次世界大战 |
|--|-------------|-------------|
| 时间 | 1914-1918 | 1939-1945 |
| 根本原因 | 帝国主义政治经济发展不平衡 | 法西斯侵略扩张 |
| 导火索 | 萨拉热窝事件 | 德国突袭波兰 |
| 战争性质 | 帝国主义战争 | 世界反法西斯战争 |
| 重要战役 | 凡尔登战役（绞肉机） | 斯大林格勒战役（转折点） |
| 结束 | 1918德国投降 | 1945日本投降 |
| 战后秩序 | 凡尔赛-华盛顿体系 | 雅尔塔体系（两极格局） |

### 冷战

| 事件 | 时间 | 说明 |
|------|------|------|
| 冷战开始 | 1947 | 杜鲁门主义出台 |
| 两极格局形成 | 1949 | 北约vs华约 |
| 冷战高潮 | 1962 | 古巴导弹危机 |
| 两极格局终结 | 1991 | 苏联解体 |

### 冷战后的世界

| 趋势 | 说明 |
|------|------|
| 政治多极化 | 美国、欧盟、日本、俄罗斯、中国等多极力量 |
| 经济全球化 | 世贸组织（WTO）、跨国公司 |
| 区域集团化 | 欧盟（EU）、亚太经合组织（APEC） |
