### 老家的甜酒酿

①每年冬天，外婆都会酿甜酒酿。糯米要提前泡一夜，蒸的时候火不能太大也不能太小。她说："火大了，米就老了，酒酿发苦；火小了，米不透心，酒酿发酸。"我常常蹲在灶边看她往蒸好的糯米里拌酒曲，那是她从老家带来的老方子，别人家都买不到。

②外婆酿酒有她自己的规矩。酒曲要用手碾碎，不能磨，她说磨出来的酒曲"不活"。拌好的糯米放在一个青花瓷罐里，罐口盖一块白棉布，再裹上一条旧棉被。冬天冷，棉被裹得越厚越好。每天她都会揭开棉被看一眼，用手指蘸一点尝尝，然后满意地点点头。

③有一天我发现柜子里整整齐齐排着十几罐酒酿，每一罐上都贴着日期。最早的一罐是去年十月，最新的是上个月。我说："外婆你酿这么多干嘛？"她笑笑没说话。

④后来我离开老家去城里上学。每次打电话回去，外婆都会问："甜酒酿吃了吗？"

⑤城里什么都能买到，但我总觉得味道不对。超市的酒酿太甜，外卖的酒酿太稀，都没有外婆做的那种——米粒饱满，酒香浓郁，咬一口，甜中带一点点酸。

⑥去年寒假回家，我对外婆说想学酿酒。外婆先是一愣，然后笑了："你呀，要学啊？来。"

⑦糯米泡好，蒸笼上锅。外婆站在一旁，没有像以前那样动手。她说："我看着你做就行了。"

⑧我笨手笨脚地拌酒曲，撒多了又倒回去一些，罐口包棉布也包得歪歪扭扭。外婆看着我，欲言又止，但最终什么都没说。

⑨半年后，我打开自己酿的第一罐酒酿。闻到那股酒香，我就好像又回到了外婆的灶台旁。

**1. 人物动态分析（10分）**

文章第②段和第⑦段都写了外婆和"我"互动的场景，但外婆的行为方式明显不同。请分析这两段中外婆的情感是否发生了变化，并说明理由。（10分）

**评分标准：** 判断情感是否变化（2分）+ 第②段分析（3分）+ 第⑦段分析（3分）+ 总结情感内核（2分）

**评分标准：** 本义（2分）+ 语境义（2分）+ 效果（3分）+ 情感/主旨（3分）

**评分标准：** 每个作用2分，答出3点即满分（6分），第4点为加分项。

### 中国古建筑中的榫卯结构

①榫卯，是古代中国建筑、家具的主要结构方式。七千多年前，河姆渡遗址中就发现了榫卯结构的木构件，这是目前已知最早的榫卯实物。

②榫卯结构的原理并不复杂：一个构件的凸出部分叫"榫"，另一个构件的凹进部分叫"卯"，榫卯相合，不用一钉一铆，就能牢固连接。

③按照连接方式的不同，榫卯结构可以分为三大类。第一类是拼板榫卯，用于将木板拼接成较宽的面板，常见于桌面、门板等；第二类是框架榫卯，用于构建家具或建筑的整体骨架，如椅子的腿和横撑之间的连接；第三类是节点榫卯，用于构件交叉处的连接，如斗拱中各个部件的结合。

④榫卯结构最神奇的特点在于它的柔性连接。当发生地震时，榫卯允许构件之间有微小位移，能够吸收部分地震能量。而铁钉连接是刚性固定，一震就断。这种柔性连接正是许多中国古建筑历经千年地震而不倒的原因。

⑤春秋战国时期，榫卯结构已广泛用于宫殿建筑。唐代以后，榫卯技术更加成熟，出现了斗拱等复杂的组合结构，使中国古建筑在世界上独树一帜。

## 四、要点总结

> 📌 **以下为本次课核心答题方法，考试前务必翻一遍。**

### 1. 词语赏析四步法（必背 ⭐⭐⭐）

```
本义 → 语境义 → 效果 → 情感
```

| 步骤 | 做什么 | 示例（"好像"） |
|------|--------|----------------|
| ① 本义 | 字典意思 | 仿佛、似乎 |
| ② 语境义 | 在文中具体指什么 | 闻到酒香仿佛回到外婆灶台旁 |
| ③ 效果 | 用这个词好在哪里 | 将现实与回忆交融 |
| ④ 情感 | 体现什么情感/主旨 | 表达对外婆的思念 |

**⚠️ 叠词专项**：四步之外多加一句"运用叠词，增强节奏感和音韵和谐"

### 2. 环境描写作用：背氛推烘（必背 ⭐⭐⭐）

| 位置 | 常考角度 | 简称 |
|------|----------|------|
| 开头 | 交代背景 + 渲染氛围 + 为下文铺垫 | **背、氛、推** |
| 结尾 | 渲染氛围 + 烘托心情 + 深化主题 | **氛、烘、主** |
| 中间 | 看和什么有关系：情节→推动，人物→烘托 | **推、烘** |

**答题口诀**：开头背氛推，结尾氛烘主，中间看关系

### 3. 人物形象动态分析三步法（必背 ⭐⭐⭐）

```
① 画时间线 —— 找出人物在不同阶段的关键行为
② 找不变的东西 —— 从头到尾有没有一个东西没变？（如：爱、关心）
③ 理解表达方式的变化 —— 同一种情感，为什么表达方式变了？
```

**核心判断**：行为不同 ≠ 情感不同！要看"表达方式"变化背后的原因

### 4. 说明顺序判断（了解）

| 看到什么词 | 判断为什么顺序 |
|-----------|----------------|
| 时间词（"早在……""后来"） | 时间顺序 |
| 方位词（"上面""前面"） | 空间顺序 |
| 都没有 | 逻辑顺序 |

**逻辑顺序常见类型**：概括→具体、原因→结果、主要→次要、整体→局部、现象→本质

> 看到"分为X类""有X种"这种总起句 → 大概率是**概括→具体**
