## 【85-105分钟】隋唐精讲（20分钟）

**动作：** 收小测，拿出草稿纸开始画时间轴。

### 1. 先串一遍大脉络（3分钟）

你在纸上画：
```
秦(前221) → 西汉(前202) → 三国两晋南北朝(220-589) → 隋(581) → 唐(618) → 五代十国 → 宋(960) → 元(1271) → 明(1368) → 清(1644)
```

你说："这是中国古代史的时间轴。学校现在讲到隋唐，今天我们先把隋唐讲扎实，然后把后面的宋元明清快速过一遍。"

### 2. 隋朝——两个"开创"（5分钟）

你说："隋朝时间很短，只有37年，但它做了两件大事，影响了一千多年。"

你在纸上写两个大字：**科举制、大运河**

**科举制：**
你说："在科举制之前，当官靠什么？靠关系，靠出身。你爸是官你就是官。科举制改了什么？——考试！凭本事当官。这打破了门第限制，普通读书人也有机会做官了。这个制度一直用到清朝末年，整整1300多年。"

> 画个箭头：魏晋"九品中正制"（看出身）→ 隋朝"科举制"（看考试）

**大运河：**
你说："隋炀帝开凿大运河，以洛阳为中心，北到涿郡（北京），南到余杭（杭州），贯通了南北。大运河的意义是什么？——**南方的粮食和物资可以运到北方**，加强了南北经济文化交流。"

### 3. 唐朝——三件大事（12分钟）

**第一件：制度完善（5分钟）**

你说："唐朝在隋朝的基础上，完善了两项制度。"

你在纸上画：

```
三省六部制：
中书省（起草）→ 门下省（审核）→ 尚书省（执行）→ 六部（吏户礼兵刑工）

科举制完善：
唐太宗增加考试科目，武则天首创武举和殿试
```

你说："三省六部制是什么意思？就是皇帝的命令不是一个人说了算的，要经过起草、审核、执行三个部门。好处是——**防止宰相权力过大，加强中央集权**。记住这个作用，考试必考。"

**第二件：盛世与治理（4分钟）**

你说："唐朝最出名的两个时期——**贞观之治**和**开元盛世**。"

| | 贞观之治 | 开元盛世 |
|--|---------|---------|
| 皇帝 | 唐太宗李世民 | 唐玄宗李隆基 |
| 特点 | 虚心纳谏（魏征）、善用人才、轻徭薄赋 | 唐朝鼎盛，经济繁荣 |

你说："贞观之治的核心词是什么？——**虚心纳谏**。唐太宗能听进去批评，这是他最厉害的地方。魏征去世他说了句话——'以铜为镜，可以正衣冠；以史为镜，可以知兴替；以人为镜，可以明得失'。这个可能考名言。"

**第三件：民族关系+对外交往（3分钟）**

你说："唐朝很开放，跟周边国家和民族关系都很好。记住几个关键词——"

你在纸上写：
- **文成公主入藏** → 嫁松赞干布，唐蕃和亲
- **鉴真东渡** → 去日本传佛教，六次才成功
- **玄奘西行** → 去天竺取经，写了《大唐西域记》
- **遣唐使** → 日本、新罗派人来唐朝学习

你说："这四个对外交往的例子要记住人物和方向。鉴真是东去日本，玄奘是西去天竺。方向别搞混。"

## 【105-115分钟】抢进度：宋元明清（10分钟）

**动作：** 时间轴上接着画。

你说："后面的朝代学校还没讲到，我快速带你过一遍，有个印象，后面学校讲的时候你就轻松了。"

### 宋朝（3分钟）

你说："宋朝最大的特点是——**经济重心彻底南移完成**。以前经济中心在北方，宋朝开始，南方经济超过北方了。证据是什么？——'苏湖熟，天下足'，苏州湖州一带的粮食丰收，全国都够吃了。"

你还说："宋朝还有三个'第一'——**最早的纸币（交子）**、**活字印刷术（毕昇）**、**指南针用于航海**。"

> 让他在时间轴考点纸上对应的宋朝部分画线标记。

### 元朝（2分钟）

你说："元朝最重要的制度——**行省制度**。就是把全国分为10个行省，每个行省派官员管理。这个制度一直用到今天，我们现在的省就是从元朝开始的。"

### 明朝（3分钟）

你说："明朝，记住一个趋势——**专制越来越强**。"

你板书：
- 废丞相 → 皇帝直接管六部
- 设锦衣卫、东厂 → 特务机构，监视官员
- 八股取士 → 考试只能按固定格式写，禁锢思想

你说："明朝对外还有一个大事——**郑和下西洋**。1405年到1433年，七次下西洋，最远到非洲东海岸。比欧洲哥伦布早了近百年，规模也大得多。"

### 清朝（2分钟）

你说："清朝专制到了顶峰——**军机处**。雍正设的，军机大臣只能跪着听皇帝旨意，只能上传下达，不能发表意见。"

你还说："清朝还有一个政策——**闭关锁国**，只开放广州一处通商。这个政策导致中国逐渐落后于世界，后面被列强打开国门就是这个原因。"
