## 二、各朝代核心考点

### 🔹 秦朝（前221-前207）

| 项目 | 内容 |
|------|------|
| 统一者 | 秦始皇嬴政 |
| 中央制度 | 皇帝制度 + 三公九卿（丞相、太尉、御史大夫） |
| 地方制度 | 郡县制（废除分封制） |
| 统一措施 | 统一文字（小篆）、统一度量衡、统一货币（圆形方孔钱） |
| 巩固措施 | 筑长城（西起临洮、东到辽东）、修灵渠 |
| 暴政 | 焚书坑儒、修阿房宫和骊山陵墓 |
| 灭亡 | 陈胜吴广起义（前209）→ 刘邦灭秦 |

---

### 🔹 西汉 + 东汉（前202-220）

| 项目 | 西汉 | 东汉 |
|------|------|------|
| 开国皇帝 | 刘邦（汉高祖） | 刘秀（光武帝） |
| 治世 | 文景之治 | 光武中兴 |
| 大一统 | 汉武帝：独尊儒术、削弱诸侯（推恩令）、北击匈奴（卫青、霍去病） | — |
| 文化 | 司马迁《史记》 | 蔡伦改进造纸术、张仲景《伤寒杂病论》 |
| 丝绸之路 | 张骞出使西域（前138、前119） | 甘英出使大秦 |

---

### 🔹 三国两晋南北朝（220-589）

| 项目 | 内容 |
|------|------|
| 三国 | 魏（曹丕）、蜀（刘备）、吴（孙权） |
| 统一 | 西晋短暂统一（280年灭吴） |
| 北方 | 北魏孝文帝改革：迁都洛阳、穿汉服、说汉语、改汉姓、通汉婚 |
| 南方 | 江南开发，经济重心开始南移 |
| 文化 | 王羲之《兰亭集序》、顾恺之《女史箴图》、祖冲之圆周率 |

---

### 🔹 隋朝（581-618）

| 项目 | 内容 |
|------|------|
| 统一 | 589年灭陈，统一南北 |
| 制度创新 | ①科举制 ②三省六部制 |
| 大工程 | 隋炀帝开凿大运河（贯通南北） |
| 灭亡 | 隋末农民起义，618年隋亡 |

---

### 🔹 唐朝（618-907）

| 项目 | 内容 |
|------|------|
| 开国 | 李渊（唐高祖）618年建国 |
| 贞观之治 | 唐太宗李世民：虚心纳谏（魏征）、善用人才、完善科举、轻徭薄赋 |
| 开元盛世 | 唐玄宗前期：唐朝鼎盛时期 |
| 制度完善 | 三省六部制（中书省→门下省→尚书省→六部）、科举制完善 |
| 经济 | 曲辕犁、筒车 |
| 民族关系 | 文成公主入藏（嫁松赞干布）、唐蕃和亲 |
| 对外交往 | 鉴真东渡日本、玄奘西行天竺（《大唐西域记》）、遣唐使 |
| 文化 | 李白（诗仙）、杜甫（诗圣）、白居易；颜真卿、柳公权书法；吴道子绘画 |
| 由盛转衰 | 安史之乱（755-763）→ 藩镇割据 → 907年灭亡 |

---

### 🔹 宋朝（960-1279）

| 项目 | 北宋 | 南宋 |
|------|------|------|
| 建立 | 赵匡胤（宋太祖）陈桥兵变 | 赵构（宋高祖）1127年 |
| 政策 | 重文轻武（杯酒释兵权） | 偏安江南 |
| 经济 | ①经济重心南移完成 ②商业繁荣（交子—最早的纸币） ③海外贸易发达 | 海外贸易继续发展 |
| 文化 | 司马光《资治通鉴》 | — |
| 科技 | 活字印刷术（毕昇）、指南针用于航海 | — |

---

### 🔹 元朝（1271-1368）

| 项目 | 内容 |
|------|------|
| 建立 | 忽必烈1271年定国号为元 |
| 政治 | 行省制度（沿用至今）、宣政院管辖西藏 |
| 经济 | 大都（北京）是繁华的商业中心 |
| 文化 | 关汉卿《窦娥冤》、马致远《天净沙·秋思》 |

---

### 🔹 明朝（1368-1644）

| 项目 | 内容 |
|------|------|
| 建立 | 朱元璋（明太祖） |
| 专制强化 | ①废除丞相 ②设锦衣卫、东厂 ③八股取士 |
| 对外 | 郑和下西洋（1405-1433，最远到非洲东海岸和红海沿岸） |
| 防御 | 戚继光抗倭 |
| 建筑 | 明长城（东起鸭绿江、西至嘉峪关）、紫禁城 |

---

### 🔹 清朝（1644-1912）

| 项目 | 内容 |
|------|------|
| 建立 | 皇太极改国号为清；1644年清军入关 |
| 专制强化 | 军机处（雍正设立）→ 君主专制达到顶峰 |
| 统一疆域 | 平定准噶尔、设驻藏大臣、台湾设行省 |
| 闭关锁国 | 只开放广州一处通商 |
| 文化 | 《红楼梦》（曹雪芹）、《四库全书》 |
| 灭亡 | 1912年溥仪退位 |
