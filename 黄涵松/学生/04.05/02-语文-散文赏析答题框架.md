## 一、词语赏析题

**常见问法：** "请赏析文中加点词语的表达效果" / "XX词好在哪里"

**答题公式：** 词语本义 + 语境义 + 表达效果（人物情感/文章主旨）

**示例：** "他用**颤抖**的手打开了信。"

> 答："颤抖"本义是身体不由自主地抖动，这里指他激动/紧张到无法控制自己。这个词生动地表现了他内心极度激动（或紧张）的状态，增强了感染力。

## 二、句子赏析题

**常见问法：** "请从修辞角度赏析画线句子" / "赏析第X段画线句"

### 第一步：判断手法

| 手法 | 判断关键词 |
|------|----------|
| 比喻 | 像、仿佛、犹如、好似 |
| 拟人 | （事物做出人的动作/情感） |
| 排比 | 三个或以上相同句式 |
| 夸张 | 大/小/多/少到不正常 |
| 反问 | 难道……？怎么……？ |
| 对偶 | 上下句字数相等、结构相同 |
| 反复 | 同一句话/词语多次出现 |

### 第二步：套公式

**比喻：** 运用了比喻的修辞手法，把__比作__，生动形象地写出了__的特点，表达了作者__的情感。

**拟人：** 运用了拟人的修辞手法，赋予__以人的__（动作/情感），生动形象地写出了__的特点，表达了作者__的情感。

**排比：** 运用了排比的修辞手法，增强了语势，层层递进地写出了__，表达了__的情感。

**反问：** 运用了反问的修辞手法，加强了语气，强调了__，表达了作者__的情感。

### 第三步：联系情感或主旨

> 想一想：作者写这句话想表达什么？思乡？怀念？感动？赞美？

## 三、主旨理解题

**常见问法：** "本文表达了作者怎样的思想感情？" / "请概括文章主旨"

**答题角度（从文章内容出发，选择合适的）：**

- 对童年/故乡/亲人的**怀念与眷恋**
- 对美好品质的**赞美与歌颂**
- 对时光流逝/物是人非的**感慨与惋惜**
- 对自然/生命的**热爱与敬畏**
- 对人生哲理的**感悟与思考**

**答题格式：** 本文通过记叙/描写____（核心事件或对象），表达了作者____（情感/主旨）。

## 四、标题/线索作用题

**常见问法：** "文章以XX为标题有什么好处？" / "XX在文中反复出现有什么作用？"

**标题作用（多选组合）：**

| 作用 | 适用情况 |
|------|---------|
| ①概括文章主要内容 | 标题直接点明核心事件/对象 |
| ②作为行文线索 | 标题中的事物在文中多次出现，贯穿全文 |
| ③揭示/暗示文章主旨 | 标题含有深层含义，需要读完才理解 |
| ④设置悬念，吸引读者 | 标题引起好奇，让人想读下去 |
| ⑤运用修辞，生动形象 | 标题用了比喻/拟人等手法 |

**线索作用：**
- 贯穿全文，使文章结构**紧凑完整**
- 推动情节发展
- 作为情感的载体，烘托/深化主题
