4．（2026·江西赣州·二模）某校九年级同学在课堂上展开了一场关于“数学与道德与法治学科融合”微讨论，产生了以下联想：

①求和——宪法是普通法律的总和

②无限循环——可随心所欲开发和利用自然资源

③最大公约数——中国共产党与各民主党派充分协商，求同存异

④一方为零，结果为零——保持身心健康是提高生命质量的关键

以上联想中，你认为合理的是（   ）

  A．①② B．①④ C．②③ D．③④

【答案】D

【难度】0.85

【知识点】爱护身体、中国共产党领导的多党合作和政治协商制度、宪法是国家的根本法、坚持绿色发展道路

【详解】①：宪法是国家的根本法，是普通法律的立法基础和立法依据，并非普通法律的简单相加总和，故①说法错误；

②：自然资源是有限的，我国坚持节约资源的基本国策，走可持续发展道路，不能随心所欲开发利用自然资源，故②说法错误；

③：我国实行中国共产党领导的多党合作和政治协商制度，核心就是寻求共识、求同存异，找到最大公约数，故③符合题意；

④：生命健康是公民享受一切权利的基础，身心健康是提高生命质量的前提和关键，失去健康，其他一切都失去意义，对应“一方为零，结果为零”，故④符合题意；

5．（2026·江西赣州·二模）校舞蹈社团选拔领舞，小安成功当选，下表内容节选于他的自荐信。小安的自荐信启示我们要（   ）

  --------------------------------------------------------------------------------------------------------------------------------------------
  我从小接受专业舞蹈教育，小学时曾担任舞蹈社团的领舞。我性格开朗，善于沟通和协调。相信我能够调动社团成员的积极性，让每次演出都取得理想效果。
  --------------------------------------------------------------------------------------------------------------------------------------------

  A．学会欣赏自己，积极展示自我

  B．参加实践活动，追逐名利回报

  C．通过自我反思，及时发现缺点

  D．重视他人评价，促进自我发展

【答案】A

【难度】0.65

【知识点】学会欣赏自己的要求

【详解】A：小安在自荐信中主动梳理介绍自己的专业能力、过往经验、性格优势，并且肯定自己可以胜任领舞工作，本质是欣赏自己的长处，抓住选拔机会积极展示自我，符合材料表达的核心，故A说法正确；

C：材料中小安只介绍了自身优势，没有涉及反思、发现自身缺点的内容，故C不符合题意；

D：材料内容是小安的自我展示，没有提到他人对小安的评价，故D不符合题意。

6．（2026·江西赣州·二模）2025年7月，李静毅携家人在西安市高陵区泾渭街道渭河河堤路游玩，忽然听到有人坠河呼救。面对险情，他毫不迟疑，纵身跃入湍急河水，奋力将落水儿童托举出水面。现场群众共同努力，成功救起落水儿童。李静毅却被水流卷走，献出了年仅34岁的生命。李静毅的行为（   ）

①践行了见义勇为、舍己为人的美德

②传递着温暖，弘扬了社会正能量

③说明生命的意义主要在于帮助他人

④表明承担责任必须付出生命代价

  A．①② B．②④ C．③④ D．①③

【答案】A

【难度】0.65

【知识点】弘扬中华传统美德、生命的意义和价值

【详解】①：李静毅不顾个人安危下水救人，践行了见义勇为、舍己为人的中华传统美德，①说法正确。

④：“必须付出生命代价”表述绝对化，承担责任往往需要付出代价，但不意味着一定要付出生命，④说法错误。

7．（2026·江西赣州·二模）他扎根一线36年，恪尽职守，心系群众冷暖，用一生践行“我愿作一颗石子”的铮铮誓言⋯⋯他就是“时代楷模”董亦军。他的事迹告诉我们（   ）

①职位高低决定人生价值的大小    

②只要兑现承诺，就能实现人生价值

③生命可以从平凡中闪耀出伟大    

④要承担社会责任，不言代价与回报

  A．①② B．①④ C．②③ D．③④

【答案】D

【难度】0.75

【知识点】承担责任的代价与回报、在平凡中创造伟大

【详解】①：人生价值的大小取决于对社会的贡献，和职位高低无关，故①说法错误；

②：“只要兑现承诺，就能实现人生价值”的表述过于绝对，实现人生价值需要为社会创造价值、做出贡献，并非仅兑现承诺就能实现，故②说法错误；

③：董亦军在平凡的一线岗位坚守多年，服务群众，体现了平凡的生命可以通过奉献闪耀出伟大，故③符合题意；

④：他扎根一线、恪尽职守心系群众，体现了主动承担社会责任，不言代价与回报的品质，故④符合题意。

③尊重顾客，买卖公道 ④文明有礼，不计利润

②：胖东来是依靠优质服务和诚信经营获得消费者认可，材料未体现垄断市场，且垄断属于不正当竞争，是违规行为，故②错误；

10．（2026·江西赣州·二模）道德与法治课上，老师展示了如下材料。这一案例体现了（   ）

+------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 案例                                                                                           | 法律规定                                                                                                                                                                         |
+------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 某公司长期拖欠员工工资，经劳动监察部门责令支付后仍拒不支付，公司负责人被判处有期徒刑并处罚金。 | 《中华人民共和国刑法》第二百七十六条之一：                                                                                                                                       |
|                                                                                                |                                                                                                                                                                                  |
|                                                                                                | 以转移财产、逃匿等方法逃避支付劳动者的劳动报酬或者有能力支付而不支付劳动者的劳动报酬，数额较大，经政府有关部门责令支付仍不支付的，处三年以下有期徒刑或者拘役，并处或者单处罚金。 |
+------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

  A．拒不支付劳动报酬只是民事违法行为

  B．违反法律的行为都要承担刑事责任

  C．公民应依法履行劳动的义务

  D．法定义务必须履行，违反法定义务须担责

【答案】D

【难度】0.65

【知识点】违法行为的含义及其分类、违反义务须承担的责任、公民必须履行法定义务的原因

【详解】A：材料中公司负责人被判处有期徒刑并处罚金，说明拒不支付劳动报酬情节严重的构成刑事犯罪，并非只是民事违法行为，故A说法错误；

B：只有违反刑法的犯罪行为才承担刑事责任，一般违法行为承担民事或行政责任，并非所有违反法律的行为都承担刑事责任，故B说法错误；

C：材料体现的是公司需履行支付劳动报酬的义务，未涉及公民劳动的义务，故C不符合题意；

D：支付劳动报酬是法定义务，公司拒不履行被依法追责，体现法定义务必须履行，违反法定义务须担责，故D符合题意。
