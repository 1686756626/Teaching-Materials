## 📝 使用说明

### 学生材料
- **原卷版**（`_原卷.md`）：直接打印给学生做，无答案
- **知识点/框架**（无后缀）：打印完整版，留作学生课后参考

### 教师材料
- **解析版**（`_解析.md`）：教师批改用，含详细解答、对比表格、做题技巧，手机/电脑查看即可
- **教案**（`_教案.md`）：逐字稿格式，可直接照着讲，包含时间分配、说的话、板书内容、过渡语
- 教案末尾附教学记录表，课后填写便于追踪学生进度

### 文件命名规范
- `XX_原卷.md` — 学生课堂做（无答案）
- `XX_解析.md` — 教师批改参考（含详解）
- `XX_要点.md` / `XX_答题框架.md` — 学生留存复习
- `XX_教案.md` — 教师用逐字稿
