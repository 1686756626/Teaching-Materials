## 📝 使用说明

### 学生材料
- 打印时**删除答案部分**（小测卷末尾的参考答案）
- 《答题框架》《知识要点》打印完整版，留作学生课后参考

### 教师教案
- 每份教案为**逐字稿**格式，可直接照着讲
- 包含：时间分配、说的话、板书内容、过渡语
- 末尾附教学记录表，课后填写便于追踪学生进度
