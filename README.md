## 📝 使用说明

### 打印清单（学生用）
原卷和要点文件直接打印，无答案，放心给学生。

### 教师参考
解析版手机/电脑上看就行，不用打印。
教案是逐字稿，打开就能讲。

### 文件命名规则

```
编号-科目-内容（类型）.md
```

| 元素 | 说明 | 示例 |
|------|------|------|
| 编号 | 两位数，按上课顺序排 | 01、02、03… |
| 科目 | 语文 / 道法 / 历史 | 语文、道法、历史 |
| 内容 | 具体知识点 | 语法小测、社会规则要点 |
| 类型 | 原卷 / 解析 / 教案 | （原卷）、（解析）、（教案） |

> 无类型后缀 = 知识要点类，打印给学生留存复习。
