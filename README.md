## 📋 文件说明

| 类型 | 定位 | 说明 |
|------|------|------|
| **学生版** | 学案 | 打印给学生，课上做题+课后带走当复习资料。不含任何答案 |
| **教师版** | 时间轴操作手册 | 题目+答案+解析+逐字稿+分支判断全部融入时间轴，老师只需这一份 |

## 📝 文件命名规则

```
编号-科目-内容（学生版）.md
编号-科目-内容（教师版）.md
```

| 元素 | 说明 | 示例 |
|------|------|------|
| 编号 | 两位数，按上课顺序排 | `01`、`02` |
| 科目 | `语文` / `道法` / `历史` | 简称，不是"道德与法治" |
| 内容 | 具体知识点或本课主题 | 期中备考冲刺 |
| 类型 | `（学生版）` / `（教师版）` | 括号包裹 |

## 📝 使用说明

### 打印清单
学生版直接打印，无答案，放心给学生。

### 教师参考
教师版是时间轴操作手册，打开就按顺序走，每道题的原文、答案、逐字稿都在里面。
