## 三、教学节奏判断

### 一模时间
- 校历期中考试约 4月16-18日（第10周）
- 一模一般与期中同步或前后脚
- **一模不一定下周，还没确定** → 不用过度赶进度，正常推进+适当备考渗透

### 学校进度 vs 一对一策略

| 情况 | 策略 |
|------|------|
| 学校正在学的 | **必须讲透**，紧跟学校节奏 |
| 学校还没到但一模会考的 | **适当抢进度**，挑核心考点速讲 |
| 一模前不可能讲完的 | 放到二轮/三轮，**不勉强** |

### 黄涵松三科进度差距
- 三科都处在一轮前半段，一模大概率覆盖不全
- 不需要焦虑，一对一优势是可以帮忙**往前抢进度**
- 长线辅导，**不用一节课解决所有问题**

## 四、课程时间安排

### 三科共用2小时
- 每科约40分钟，必须**抓重点、做取舍**
- 不要每科都讲新内容，留5分钟做小测摸底
- 每节课最多安排：小测 + 1个核心知识点 + 适当抢进度

### 2小时课的标准结构

| 时段 | 内容 | 时长 |
|------|------|:----:|
| 小测 | 5道题快速摸底 | 5min |
| 核心 | 紧跟学校的知识点精讲 | 15-20min |
| 休息 | ☕ | 5min |
| 抢进度 | 适当往前带1-2个知识点 | 10-15min |
| 练习 | 做题巩固 | 5-10min |
| 总结 | 布置作业 | 2-3min |
